# Responsibility

This node owns repository-level licensing metadata for Musicon.

It is responsible for declaring the distribution terms that apply to the source tree and shipped artifacts so contributors and downstream users can determine their rights and obligations from the repository itself.

# Boundaries

This node is not responsible for CI execution, release packaging mechanics, or application runtime behavior.

It records the legal terms under which the repository is published, not the operational systems that build or run the software.
